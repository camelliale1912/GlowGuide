# Installation and how to run
1. Create and activate env\
```bash
python3 -m venv .venv
```

2. Activate environment
- macOS and Linux
```bash
source .venv/bin/activate
```
- Windows
```bash
.venv\Scripts\activate
```

3. Install requirements
```bash
pip install -r requirements.txt
```

4. Run the app
```bash
streamlit run app.py
````
